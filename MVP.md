# SVX MVP

## Objective

Enable a user to store, track, and retrieve an asset.

---

## Required Features

### Asset Creation

- Asset Name
- Photos
- Description
- Estimated Value

---

### Custody Selection

- Select storage location
- Request custody

---

### Chain of Custody

- Asset ID
- Intake timestamp
- Release timestamp
- Custodian record

---

### Tracking

- Current status
- Current custodian
- Custody history

---

### Retrieval

- Request asset return
- Verify ownership
- Release asset

---

## Not Included

- Banking integrations
- Advanced insurance
- Blind routing
- Multi-location transfers
- Enterprise features

These can be added later.
